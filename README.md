This is the frontend application of the [Archives](https://archives.opendata.lk). This is a vite + react applicaiton. Refer to the backend from here [Archives- Bakcend](https://github.com/LDFLK/gztarchiver-ui-backend). Which you can explore government documents. and interact with the documents.

## Features


| Feature | Description |
|---------|-------------|
| **Document Stat Dashboard** | Real-time overview of the total archived documents, covered years, and available document types with visual statistics. |
| **Advanced Search** | Powerful search engine with support for exact phrase matching and filters by date, type, ID, availability, and source. |
| **Document Relationship Explorer** | Interactive graph visualization to explore connections, amendments, and references between different specific documents. |
| **Play with documents** | Interactive features like "Take me somewhere" to discover random documents and explore the archive serendipitously. |
| **Document Viewer** | Seamless integration to view metadata and access document content directly from the search results. |
| **Document Downloader** | Direct download capability to access the original archived files in their native formats. |
